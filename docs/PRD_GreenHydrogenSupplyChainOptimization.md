## 1. 项目概述 | Project Overview

### 1.1 项目背景

基于现有的天然气供应链优化系统（`natural_gas_supply_chain_optimization`），开发一个全新的**绿氢供应链优化系统**，用于优化绿氢结合CO₂捕获生产可持续航空燃料(SAF)的供应链决策。

**核心变化：**
- **原工艺路线**: 天然气 → SAF（一步法）或 天然气+绿氢 → 甲醇 → SAF（两步法）
- **新工艺路线**: 绿氢+CO₂ → SAF（费托合成一步法 或 甲醇中间路径两步法）

### 1.2 项目目标

1. **技术目标**
   - 建立绿氢+CO₂制SAF的供应链优化模型
   - 实现CO₂捕获源的地理定位与捕获量计算
   - 优化氢气运输、CO₂运输和SAF生产的综合成本
   - 支持两种技术路径的对比分析（费托合成 vs 甲醇路径）

2. **经济目标**
   - 最小化总供应链成本（CAPEX + OPEX）
   - 计算平准化成本（LCOE）并与传统路径对比
   - 评估不同技术路径的经济可行性

3. **环境目标**
   - 全生命周期碳排放评估
   - CO₂利用效率分析
   - 实现负碳或净零排放SAF生产

### 1.3 与现有项目的关系

- **代码复用**: 复用现有的优化框架、路径规划引擎、缓存管理等模块
- **架构继承**: 继承模块化设计、数据流分离、结果分类等架构原则
- **差异化**: 核心优化模型完全重构，适应新的工艺路线和约束条件
