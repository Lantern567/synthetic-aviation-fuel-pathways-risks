# 天然气供应链MTJ航煤生产优化模型 - 平准化成本参数表

本文档详细列出了优化模型中使用的所有成本参数，**现已更新为基于固定成本+可变成本的规模经济成本结构**。

## 1. 新成本结构设计理念

### 1.1 规模经济成本模型
```
投资成本 = 固定投资成本 + 可变投资成本 × 产能
运营成本 = 固定运营成本 + 可变运营成本 × 产量
```

### 1.2 关键改进点
- **固定成本体现门槛效应**：基础设施、控制系统等一次性投入
- **可变成本体现规模效应**：反应器、分离设备等随产能线性增长
- **避免成本过高**：通过合理拆分降低投资门槛
- **激励建厂决策**：提高缺货惩罚，使建厂更经济

## 2. MTJ生产设施投资成本

### 2.1 固定投资成本（每个工厂）
```
固定投资成本 = 2,000,000元（200万元）
年化系数 = 12%
固定投资年化成本 = 200万 × 12% = 24万元/年
```

**固定投资包含**：
- 土建工程：厂房、基础设施
- 控制系统：自动化控制设备
- 公用工程：水电汽系统基础部分
- 安装调试：基础安装费用
- 环保设施：基础环保设备

### 2.2 可变投资成本（随产能变化）
```
可变投资成本 = 8,000元/(kg/hour)
年化系数 = 12%
可变投资年化成本 = 8000 × 产能 × 12% 元/年
```

**可变投资包含**：
- F-T反应器：按产能配置
- 分离单元：产能相关设备
- 加氢精制单元：处理能力相关
- 换热器网络：随产能扩展
- 压缩机：功率随产能增长

### 2.3 投资成本对比
| 工厂规模 | 固定投资年化 | 可变投资年化 | 总投资年化 | 单位投资成本 |
|----------|--------------|--------------|------------|--------------|
| 500 kg/h | 24万元/年 | 48万元/年 | 72万元/年 | 1440元/(kg/h)/年 |
| 1000 kg/h | 24万元/年 | 96万元/年 | 120万元/年 | 1200元/(kg/h)/年 |
| 2000 kg/h | 24万元/年 | 192万元/年 | 216万元/年 | 1080元/(kg/h)/年 |

**规模经济效应**：单位投资成本随规模增大而降低

## 3. MTJ生产运营成本

### 3.1 固定运营成本（每个工厂每年）
```
固定运营成本 = 1,000,000元/年（100万元/年）
```

**固定运营包含**：
- 人工成本：操作人员基础工资
- 管理费用：工厂管理运营费用  
- 基础维护：定期保养、检修
- 保险费用：设备保险
- 其他固定开支：办公、通讯等

### 3.2 可变运营成本（随产量变化）
```
可变运营成本 = 300元/kg产品
```

**可变运营包含**：
- 催化剂消耗：150元/kg（50%）
- 电耗费用：80元/kg（27%）  
- 设备磨损维护：50元/kg（17%）
- 化学品消耗：20元/kg（6%）

### 3.3 运营成本对比
| 年产量 | 固定运营 | 可变运营 | 总运营成本 | 单位运营成本 |
|--------|----------|----------|------------|--------------|
| 100万kg | 100万元 | 3000万元 | 3100万元 | 310元/kg |
| 500万kg | 100万元 | 15000万元 | 15100万元 | 302元/kg |
| 800万kg | 100万元 | 24000万元 | 24100万元 | 301元/kg |

**规模经济效应**：单位运营成本随产量增大而降低

## 4. 技术模式复杂度调整

### 4.1 基础成本计算
基于1000kg/h工厂，85%容量因子：
- 年产量：1000 × 8760 × 0.85 = 744.6万kg/年
- 投资年化：24 + 96 = 120万元/年
- 固定运营：100万元/年
- 可变运营：300 × 744.6万 = 22338万元/年
- **基础总成本**：22558万元/年
- **基础单位成本**：22558万 ÷ 744.6万 = 303元/kg

### 4.2 复杂度因子应用
各技术模式在基础成本上应用复杂度调整：

| 技术模式 | 复杂度因子 | 调整后成本 | 技术特点 |
|----------|------------|------------|----------|
| LNG接收站MTJ生产 | 0.85 | **258元/kg** | 港口集成，规模经济优势 |
| 管段直供MTJ生产 | 0.90 | **273元/kg** | 最简单，就地制氢 |
| LNG转运MTJ生产 | 1.00 | **303元/kg** | 标准复杂度基准 |
| 机场集成MTJ生产 | 1.15 | **348元/kg** | 机场环境限制 |
| 综合供应MTJ生产 | 1.30 | **394元/kg** | 最复杂，多源集成 |

## 5. 缺货惩罚成本调整

### 5.1 惩罚成本设定

```
缺货惩罚成本 = 2,500元/kg
```

### 5.2 经济性对比分析
| 决策选择 | 成本计算 | 总成本估算 |

## 9. 成本结构优化效果

### 9.1 优化前后对比
| 项目 | 优化前 | 优化后 | 改进效果 |
|------|--------|--------|----------|
| 投资门槛 | 单一高成本 | 固定+可变结构 | 降低进入门槛 |
| 规模经济 | 无体现 | 明显体现 | 鼓励大规模建设 |
| 建厂激励 | 缺货更便宜 | 建厂更经济 | 模型倾向建厂 |
| 成本合理性 | 过高 | 合理 | 符合工业实际 |

### 9.2 经济激励机制
1. **固定成本摊销**：大规模工厂摊薄固定成本
2. **缺货惩罚驱动**：2500元/kg惩罚远高于生产成本
3. **技术选择优化**：复杂度因子引导最优技术选择
4. **区域布局优化**：考虑运输成本的空间配置
