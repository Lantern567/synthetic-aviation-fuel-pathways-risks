# 变更日志

本文档记录航空港数据处理项目的所有重要变更。

## [1.1.0] - 2024-12-20

### 新增功能
- **燃油消耗计算模块**: 基于航班数据计算每趟航班的燃油消耗
  - 支持中文机型到ICAO代码的自动映射
  - 考虑载客数对燃油消耗的影响
  - 实现距离影响系数计算
  - 支持多种机型的燃油消耗估算

- **机型映射模块** (`aircraft_mapping.py`)
  - 中文机型名称到ICAO代码转换
  - 机型典型载客量数据库
  - 载客率计算功能
  - 机型性能参数定义

- **综合计算模块** (`integrated_fuel_calculator.py`)
  - 优先使用pybada库进行精确计算
  - 自动回退到经验公式方法
  - 支持计算方法对比分析

- **演示模块** (`demo_fuel_calculator.py`)
  - 小批量数据快速验证
  - 详细的统计报告生成
  - 按机型分组的燃油消耗分析

### 测试覆盖
- **单元测试** (`test_fuel_calculator.py`)
  - 机型映射功能测试
  - 载客率计算测试
  - 燃油消耗计算验证
  - 边界条件测试
  - 样例计算演示

### 支持的机型
- 波音系列: B737, B757, B777, B787
- 空客系列: A319, A320, A321, A330, A380
- 其他机型: ERJ-190, CRJ900, CRJ200等

### 计算特性
- **多因素考虑**: 机型、载客数、飞行距离
- **智能映射**: 自动处理中文机型名称
- **批量处理**: 支持大数据文件分批处理
- **结果验证**: 完整的单元测试覆盖
- **详细报告**: Excel格式的多工作表输出

### 依赖库
- **必需**: pandas, numpy, xlsxwriter
- **可选**: pyBADA (用于高精度BADA模型计算)

### 输出格式
生成Excel文件包含：
1. 航班燃油消耗详细数据
2. 统计汇总信息
3. 按机型分组的统计报告

## 技术说明

### 燃油消耗计算公式
基础燃油消耗 = 距离 × 机型基础消耗率 × 载客率系数 × 距离系数

### 载客率影响
载客率系数 = 0.7 + 0.3 × 载客率
(载客率越高，分摊效率越好)

### 距离影响
- 短距离(<500km): 系数1.3 (起降燃油占比高)
- 中距离(500-1500km): 系数1.0 (标准)
- 长距离(>1500km): 系数0.95 (巡航效率高)
