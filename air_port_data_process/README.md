## 计算方法

### 燃油消耗公式
```
燃油消耗(kg) = 距离(km) × 机型基础消耗率 × 载客率系数 × 距离系数 × 燃油密度
```

### 关键参数
- **载客率系数**: 0.7 + 0.3 × 载客率
- **距离系数**: 
  - 短程(<500km): 1.3
  - 中程(500-1500km): 1.0  
  - 长程(>1500km): 0.95
- **燃油密度**: 0.8 kg/L

### 距离修复策略 🆕
1. **Haversine公式**: 基于城市/机场坐标计算球面距离
2. **城市数据库**: 常见城市间距离估算
3. **智能默认**: 根据城市类型分配合理距离
