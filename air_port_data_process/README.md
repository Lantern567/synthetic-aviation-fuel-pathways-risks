# 航空港数据处理项目

本项目用于处理和分析航空港相关数据，包括航班数据处理、燃油消耗计算等功能。

## 项目结构

```
air_port_data_process/
├── data/                           # 数据文件目录
│   ├── 22年1月1日至24年12月31日航班数据.xlsx
│   └── ...
├── src/                            # 源代码目录
│   ├── aircraft_mapping.py        # 航空器机型映射模块
│   ├── fuel_consumption_calculator.py  # 燃油消耗计算模块
│   ├── demo_fuel_calculator.py     # 演示版燃油消耗计算
│   ├── integrated_fuel_calculator.py   # 综合燃油消耗计算模块
│   └── flight_fuel_calculator.py  # 原始燃油计算器
├── tests/                          # 测试代码目录
│   └── test_fuel_calculator.py     # 燃油计算功能测试
├── results/                        # 结果输出目录
│   ├── tables/                     # 表格结果
│   ├── figures/                    # 图表结果
│   └── reports/                    # 报告结果
└── logs/                          # 日志目录
```

## 主要功能

### 1. 航班燃油消耗计算

本项目实现了基于航班数据计算燃油消耗的功能，考虑以下因素：

- **机型影响**: 支持中文机型到ICAO代码的自动映射
- **载客数影响**: 根据实际载客数计算载客率，影响燃油消耗
- **距离影响**: 考虑飞行距离对燃油效率的影响
- **多种计算方法**: 支持pybada库（如可用）和经验公式两种计算方法

#### 支持的机型

| 中文机型 | ICAO代码 | 典型载客量 |
|---------|----------|-----------|
| 波音737(中) | B737 | 160 |
| 空客320(中) | A320 | 160 |
| 空客321(中) | A321 | 200 |
| 波音777(大) | B777 | 350 |
| 空客330(宽体机) | A330 | 290 |
| ... | ... | ... |

### 2. 核心模块说明

#### aircraft_mapping.py
- 中文机型到ICAO代码映射
- 机型载客量和性能参数定义
- 载客率计算功能

#### fuel_consumption_calculator.py
- 基于经验公式的燃油消耗计算
- 批量数据处理功能
- 考虑机型、载客数、距离等因素

#### integrated_fuel_calculator.py
- 综合燃油消耗计算模块
- 优先使用pybada库，回退到经验公式
- 支持多种计算方法对比

## 使用方法

### 环境要求

```bash
# 必需依赖
pip install pandas numpy xlsxwriter

# 可选依赖（用于高精度计算）
pip install pyBADA
```

### 基本使用

```python
# 演示版本（处理小批量数据）
python src/demo_fuel_calculator.py

# 综合计算（推荐）
python src/integrated_fuel_calculator.py

# 完整数据处理
python src/fuel_consumption_calculator.py
```

### 测试

```bash
# 运行单元测试
python tests/test_fuel_calculator.py
```

## 计算结果

计算完成后，将在 `results/tables/` 目录下生成Excel文件，包含：

1. **航班燃油消耗工作表**: 原始数据 + 新增字段
   - ICAO代码: 转换后的机型代码
   - 载客率: 实际载客数/典型载客量
   - 燃油消耗_kg: 计算得出的燃油消耗量
   - 计算方法: 使用的计算方法（pybada/empirical）
   - 计算状态: 计算是否成功

2. **统计汇总工作表**: 整体统计信息
   - 总航班数、成功计算数、成功率
   - 总燃油消耗、平均燃油消耗
   - 平均载客率等

3. **机型统计工作表**: 按机型分组的统计
   - 各机型的航班数量和燃油消耗情况

## 计算方法

### 经验公式方法

基于航空业界经验数据，考虑：
- 机型基础燃油消耗率（升/公里）
- 载客率影响系数
- 距离影响系数（短距离效率较低）
- 起降额外燃油消耗

### pybada方法（如可用）

基于EUROCONTROL的BADA模型：
- 更精确的气动和性能参数
- 考虑飞行高度、温度等因素
- 标准的航空器性能建模
