# Examples Directory

这个目录包含绿色甲醇项目的代码模式和示例，在实现新功能时应该遵循这些模式。

## 项目代码模式示例

### 1. 数据处理模式 (data_processing_pattern.py)
展示标准的科学计算和数据处理流程：
- **数据读取**: pandas支持多种格式（CSV, Excel, JSON）
- **数据清理**: 缺失值处理、异常值检测、数据类型转换
- **数据验证**: 完整性检查、业务规则验证、质量控制
- **结果保存**: 时间戳命名、分类存储、元数据记录
- **错误处理**: 异常捕获、日志记录、优雅降级

**使用场景**: 所有数据密集型功能都应遵循此模式

### 2. 测试模式 (test_pattern.py)
完整的测试框架和验证模式：
- **单元测试**: pytest框架、断言模式、测试隔离
- **数据测试**: 测试数据生成、Mock对象使用、边界条件
- **集成测试**: 端到端流程验证、系统集成测试
- **性能测试**: 执行时间测量、内存使用监控
- **回归测试**: 结果一致性验证、版本兼容性

**使用场景**: 每个新功能都必须包含相应的测试

### 3. 可视化模式
基于项目中现有的可视化实现：
- **静态图表**: matplotlib + seaborn，中文字体支持
- **交互式地图**: pydeck + Folium，地理数据可视化
- **仪表板**: HTML报告生成，多图表集成
- **颜色方案**: 项目统一的颜色配置
- **输出格式**: PNG高分辨率 + HTML交互版本

**参考文件**: `products/aviation_fuel_analysis/airline_visualization/`

## 架构模式

### 4. 模块组织模式
基于PBR架构的标准模块结构：
```
module_name/
├── src/                    # 源代码
│   ├── data_loader.py      # 数据加载器
│   ├── processor.py        # 核心处理逻辑
│   ├── calculator.py       # 计算引擎
│   └── visualizer.py       # 可视化组件
├── data/                   # 输入数据
├── results/                # 输出结果
│   ├── tables/            # 表格数据
│   ├── figures/           # 图表
│   └── reports/           # 报告
├── tests/                  # 测试文件
└── logs/                   # 日志记录
```

### 5. 算法集成模式
科学计算和优化算法的标准集成：
- **优化求解器**: Gurobi数学模型构建和求解
- **地理计算**: GraphHopper路径规划集成
- **物理建模**: pyBADA航空物理计算
- **并行计算**: 多进程/多线程处理大数据集
- **缓存机制**: 计算结果缓存和重用

**参考文件**: `products/supply_chain_optimization/natural_gas_supply_chain_optimization/`

## 领域特定模式

### 6. 能源分析模式
能源系统分析的专业模式：
- **成本建模**: 平准化成本计算、敏感性分析
- **环境影响**: 碳排放计算、生命周期评估
- **供应链建模**: 多阶段优化、约束条件处理
- **不确定性分析**: 蒙特卡洛模拟、情景分析

### 7. 地理信息模式
GIS数据处理和空间分析：
- **坐标系统**: WGS84标准化、投影变换
- **空间查询**: 距离计算、区域分析、路径优化
- **地图渲染**: 多层地图、交互式控件
- **数据融合**: 多源地理数据集成

**参考文件**: `products/gis_energy_mapping/`
