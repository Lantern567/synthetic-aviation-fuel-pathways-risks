# Examples Directory

This directory contains code patterns and examples from the green methanol project that should be followed when implementing new features.

## 项目代码模式示例

### 数据处理模式 (data_processing_pattern.py)
- pandas数据读取和预处理标准模式
- Excel/CSV文件处理范例
- 错误处理和数据验证模式

### 可视化模式 (visualization_pattern.py)
- matplotlib静态图表创建模式
- 中文字体设置和颜色方案
- 图表保存和输出格式标准

### 测试模式 (test_pattern.py)
- pytest单元测试结构
- 测试数据准备和清理
- 模拟数据生成和验证方法

### 优化计算模式 (optimization_pattern.py)
- Gurobi优化模型设置范例
- 大规模数据并行处理模式
- 性能监控和资源管理

### 日志记录模式 (logging_pattern.py)
- Python logging配置标准
- 日志文件命名和轮转
- 错误跟踪和调试信息记录

### GIS数据处理模式 (gis_pattern.py)
- GeoJSON数据读取和处理
- 地理坐标转换和验证
- 空间数据可视化模式

## 使用说明

1. **查看模式**：在实现新功能前，先查看相关的模式示例
2. **遵循规范**：严格按照示例中的结构和命名规范
3. **适应性修改**：根据具体需求适当调整，但保持核心模式不变
4. **更新示例**：发现更好的实现模式时，及时更新示例文件
