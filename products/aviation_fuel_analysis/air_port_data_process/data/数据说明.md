# 数据文件说明

## 航班数据文件

### 原始数据文件
- **文件名**: `22年1月1日至24年12月31日航班数据.xlsx`
- **文件大小**: 约190MB
- **数据范围**: 2022年1月1日至2024年12月31日
- **记录数量**: 约数十万条航班记录

### 数据字段

原始数据包含以下字段：
- 出发城市、出发城市坐标(x,y)
- 到达城市、到达城市坐标(x,y)  
- 里程（公里）
- 航班班次
- 航空公司
- **机型** (燃油消耗计算的关键字段)
- 起飞机场、起飞机场坐标(x,y)
- 降落机场、降落机场坐标(x,y)
- 准点率、平均误点时间
- 每周班期信息
- 出发/到达省份
- 起飞/降落时间
- 日期
- 价格(元)
- **人数** (燃油消耗计算的关键字段)

### 数据使用说明

由于原始数据文件过大（>100MB），已被.gitignore排除。如需使用完整数据：

1. **获取数据文件**: 请从原始数据源获取Excel文件
2. **放置位置**: 将文件放在本目录下，命名为 `22年1月1日至24年12月31日航班数据.xlsx`
3. **运行计算**: 运行燃油消耗计算脚本即可自动读取数据

### 处理能力

项目代码支持：
- 小批量处理：200-500条记录（演示用）
- 分批处理：适合处理完整数据集
- 内存优化：避免一次性加载所有数据

### 输出结果

计算完成后将生成：
- 详细的燃油消耗数据表
